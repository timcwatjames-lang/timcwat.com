1. What is CSS?

CSS (Cascading Style Sheets) controls the visual presentation of HTML elements — colors, layout, fonts, spacing, and more. CSS rules have a selector and a declaration block of property-value pairs.

〈 CSS Syntax
<!-- HTML -->
<p id="greeting">Hello, World!</p>

/* CSS */
#greeting {
    color: #667eea;
    font-size: 24px;
    font-weight: 700;
    padding: 10px 20px;
    background: #f0f4ff;
    border-radius: 8px;
}
▶ Output

Hello, World!

Three Ways to Add CSS

〈 Inline, Internal, External
<!-- 1. Inline (on the element) -->
<p style="color: red;">Red text</p>

<!-- 2. Internal (<style> in <head>) -->
<style>
    p { color: blue; }
</style>

<!-- 3. External (.css file) -->
<link rel="stylesheet"
      href="style.css">
▶ Result

Red (inline style)

Blue (internal style)

Green (would be from external)

💡 Key idea: Use external CSS for real projects. It keeps presentation separate from content and allows caching. The "C" in CSS stands for Cascading — styles can inherit, override, and layer from multiple sources.

2. Selectors

Selectors target HTML elements to apply styles. From simple tag selectors to complex combinators, CSS gives you fine-grained control over what gets styled.

〈 Basic Selectors
/* Element / Tag */
h2 { color: #333; }

/* Class */
.highlight { background: #ffeb3b; }

/* ID */
#special { border: 2px solid gold; }

/* Universal */
* { box-sizing: border-box; }

/* Grouping */
h1, h2, h3 {
    font-family: sans-serif;
}

〈 Combinators
/* Descendant (space) */
article p { color: #555; }

/* Child (>) */
.list > li {
    font-weight: 600;
}

/* Adjacent sibling (+) */
h3 + p {
    margin-top: 0;
}

/* Attribute */
a[href^="https"] {
    color: green;
}

5. Box Model

Every element is a rectangular box. The box model describes how content, padding, border, and margin layers stack up.

〈 Box Model
.box-demo {
    width: 200px;
    height: 100px;
    padding: 20px;   /* inside */
    border: 4px solid #333;
    margin: 20px;   /* outside */
    box-sizing: border-box;
}

/* content-box (default) */
/* width = content only */

💡 Best practice: Always set box-sizing: border-box on all elements. It makes sizing much more intuitive. Most reset CSS files do this globally with * { box-sizing: border-box; }.

6. Display & Positioning

The display property controls how elements flow in the document. position controls where they sit on the page.

Display Values

〈 Display
/* Block: takes full width */
.block { display: block; }

/* Inline: sits in text flow */
.inline { display: inline; }

/* Inline-block: inline but */
/* respects w/h, margin */
.ib {
    display: inline-block;
    width: 120px;
}

/* None: removes from flow */
.hidden { display: none; }

Position Values

〈 Position
/* Static: default flow */
.static { position: static; }

/* Relative: offset from normal */
.relative {
    position: relative;
    top: 10px; left: 10px;
}

/* Absolute: relative to parent */
.absolute {
    position: absolute;
    top: 0; right: 0;
}

/* Fixed: relative to viewport */
.fixed {
    position: fixed;
    bottom: 20px;
    right: 20px;
}

/* Sticky: toggles relative/fixed */
.sticky {
    position: sticky;
    top: 0;
}

7. Flexbox

Flexbox is a one-dimensional layout system designed for distributing space and aligning items in rows or columns.

〈 Flexbox Container
.flex-container {
    display: flex;
    flex-direction: row;
    justify-content:
        center;
    align-items: center;
    gap: l2px;
    flex-wrap: wrap;
}

/* Item properties */
.flex-item {
    flex: 1;  /* grow + shrink */
}

8. CSS Grid

Grid is a two-dimensional layout system that handles rows and columns simultaneously. Perfect for page-level layouts and complex alignments.

〈 Grid Basics
.grid-container {
    display: grid;
    grid-template-columns:
        repeat(3, 1fr);
    gap: 12px;
}

/* Span multiple columns */
.grid-item {
    grid-column: span 2;
}

9. Responsive Design

Responsive design makes your site look good on all screen sizes. Key tools: media queries, relative units, and flexible layouts.

〈 Media Queries
/* Mobile-first approach */

/* Base: mobile styles */
.card {
    padding: 1rem;
    font-size: 0.9rem;
}

/* Tablet (768px+) */
@media (min-width: 768px) {
    .card {
        padding: 2rem;
        font-size: 1rem;
    }
}

/* Desktop (1024px+) */
@media (min-width: 1024px) {
    .card {
        padding: 3rem;
    }
}

Common Relative Units

〈 Units
width: 50%;         /* % of parent */
font-size: 1.5rem;    /* relative to root (16px default) */
padding: 2em;         /* relative to this element's font-size */
width: 100vw;        /* viewport width */
height: 100vh;       /* viewport height */
font-size: clamp(1rem, 2.5vw, 2rem); /* fluid */

10. Transitions & Animations

Transitions smoothly change property values over time. Animations use @keyframes for more complex, multi-step sequences.

Transitions

〈 Transitions
.btn-hover {
    background: #667eea;
    color: #fff;
    padding: 12px 24px;
    border-radius: 8px;
    transform: scale(1);
    transition:
        all 0.3s ease;

.btn-hover:hover {

    transform: scale(1.05);
    box-shadow:

11. Pseudo-classes & Pseudo-elements

Pseudo-classes (one colon) target elements in a specific state. Pseudo-elements (two colons) target specific parts of an element.

〈 Pseudo-classes
/* State-based */
a:hover {
    color: #764ba2;
}
input:focus {
    border-color: #667eea;
    outline: none;
}

/* Position-based */
li:first-child {
    font-weight: 700;
}
li:last-child {
    border: none;
}
li:nth-child(odd) {
    background: #f0f4ff;
}

/* Negation */
p:not(.special) {
    color: #666;
}
▶ Demo

  • nth-child(odd) bg

  • last-child (no border)
〈 Pseudo-elements
/* Before / After */
.quote::before {
    content: "\201C";  /* open quote */
    font-size: 2rem;
    color: #667eea;
}

.quote::after {
    content: "\201D";  /* close quote */
}

/* First letter / line */
p::first-letter {
    font-size: 2em;
    font-weight: 700;
    color: #667eea;
}

/* Selection highlight */
::selection {
    background: #667eea;
    color: #fff;
}

12. Advanced CSS

Modern CSS includes powerful features like custom properties (variables), calc(), clamp(), filters, transforms, and layered shadows.

〈 Custom Properties
:root {
    --primary: #667eea;
    --secondary: #764ba2;
    --radius: 12px;
    --shadow:
        0 4px 20px
        rgba(102,126,234,0.3);
}

.card {
    background:
        var(--primary);
    border-radius:
        var(--radius);
    box-shadow:
        var(--shadow);
}

Filters & Transforms

〈 Filters & Transforms
/* Filters */
.blur {
    filter: blur(2px);
}
.grayscale {
    filter:
        grayscale(100%);
}

/* Transforms */
.rotated {
    transform:
        rotate(15deg)
        scale(1.1);
}

/* calc() */
.fluid {
    width:
        calc(100% - 40px);
}
