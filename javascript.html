1. What is JavaScript?

JavaScript is a lightweight, interpreted programming language that makes web pages interactive. It runs in the browser and can respond to user actions, modify content, and communicate with servers.

〈 Your First JS
console.log("Hello, World!");
console.log("Welcome to JavaScript!");
console.log("This is " + "fun!");

💡 Key idea: JavaScript is event-driven. It waits for something to happen (a click, a key press, a page load) and then runs code in response. You can also run it directly in the browser's console (press F12).

2. Variables & Data Types

Variables store data in memory. Use let for values that change, const for values that stay the same, and avoid var in modern code.

console.log("Age:", age);
console.log("Student:", isStudent);

// Reassign let (works)
name = "Bob";
console.log("New name:", name);

3. Operators

Operators perform actions on values. JavaScript has arithmetic, comparison, logical, and assignment operators.

Arithmetic Operators

〈 Math
let a = 10, b = 3;

console.log("a + b =", a + b);  // 13
console.log("a - b =", a - b);  // 7
console.log("a * b =", a * b);  // 30
console.log("a / b =", a / b);  // 3.33
console.log("a % b =", a % b);  // 1 (remainder)
console.log("a ** b =", a ** b); // 1000 (power)
▶ Output
// Click Run to execute

Comparison & Logical Operators

〈 Comparisons
console.log("5 == '5' :", 5 == '5');   // true (loose)
console.log("5 === '5':", 5 === '5');  // false (strict)
console.log("10 > 5   :", 10 > 5);       // true
console.log("10 <= 5  :", 10 <= 5);      // false

// Logical: && (AND), || (OR), ! (NOT)
console.log("true && false:", true && false);  // false
console.log("true || false:", true || false);  // true
console.log("!true       :", !true);              // false

4. Functions

Functions are reusable blocks of code. You define them once and call them anytime. They can take inputs (parameters) and return outputs.

Function Declaration

〈 Functions
// Function declaration
function greet(name) {
    return "Hello, " + name + "!";
}

// Arrow function (modern)
const add = (a, b) => a + b;

console.log(greet("Alice"));
console.log("2 + 3 =", add(2, 3));

// Function with default parameter
const multiply = (a, b = 1) => a * b;
console.log(multiply(5));      // 5 (b defaults to 1)
console.log(multiply(5, 3));   // 15

5. Conditionals

Conditionals let your code make decisions. Run different code depending on whether a condition is true or false.

if / else if / else

〈 Conditionals
let score = 85;

if (score >= 90) {
    console.log("Grade: A");
} else if (score >= 80) {
    console.log("Grade: B");
} else if (score >= 70) {
    console.log("Grade: C");
} else {
    console.log("Grade: F");
}

// Ternary operator (shorthand if/else)
let age = 20;
let status = age >= 18 ? "Adult" : "Minor";
console.log("Status:", status);

7. Arrays

Arrays hold ordered lists of data. You can store any type — numbers, strings, objects, even other arrays.

Array Methods

〈 Array Methods
let numbers = [1, 2, 3, 4, 5];

console.log("Original:", numbers);
console.log("Length:", numbers.length);
console.log("First:", numbers[0]);
console.log("Last:", numbers[numbers.length - 1]);

// Add/remove
numbers.push(6);    // add to end
console.log("After push:", numbers);
numbers.pop();     // remove from end
console.log("After pop:", numbers);

// Transform
let doubled = numbers.map(n => n * 2);
console.log("Doubled:", doubled);

let evens = numbers.filter(n => n % 2 === 0);
console.log("Evens:", evens);

let sum = numbers.reduce((acc, n) => acc + n, 0);
console.log("Sum:", sum);

8. Objects

Objects store key-value pairs. They're the foundation of most data structures in JavaScript and represent real-world entities.

〈 Objects
// Creating an object
const person = {
    name: "Alice",
    age: 30,
    city: "Lagos",
    greet: function() {
        return "Hi, I'm " + this.name;
    }
};

console.log("Name:", person.name);
console.log("Age:", person["age"]);
console.log(person.greet());

// Add/edit properties
person.job = "Developer";
console.log("Job:", person.job);

// List all keys
console.log("Keys:", Object.keys(person));

9. DOM Manipulation

The Document Object Model (DOM) is JavaScript's way of interacting with HTML. You can select elements, change their content, modify styles, and create new elements.

〈 DOM Basics
// Select an element
const title = document.getElementById("dom-title");

// Change content
title.textContent = "Updated by JS!";

// Change style
title.style.color = "#667eea";

// Create an element
const newP = document.createElement("p");
newP.textContent = "I was created by JavaScript!";
document.getElementById("dom-container")
        .appendChild(newP);
▶ Output

This is the original text.

💡 Common DOM methods: getElementById(), querySelector(), querySelectorAll(), createElement(), textContent, innerHTML, setAttribute(), classList.add(), classList.remove(), style.

10. Events

Events are actions that happen in the browser — clicks, key presses, mouse movements, form submissions. You can run code when events occur.

Click & Input Events

〈 Events
// Click event
document.getElementById("myBtn")
    .addEventListener("click", () => {
        alert("Button clicked!");
    });

// Input event
document.getElementById("myInput")
    .addEventListener("input", (e) => {
        console.log("You typed:", e.target.value);
    });

// Mouseover event
document.getElementById("hoverBox")
    .addEventListener("mouseover", () => {
        console.log("Mouse entered!");
    });

Hover over this box
💡 Event types: click, dblclick, mouseover, mouseout, keydown, keyup, input, change, submit, scroll, resize, load. Use addEventListener() to attach them.

11. ES6+ Features

ES6 (ECMAScript 2015) and later versions added powerful features that make JavaScript cleaner and more expressive.

〈 Modern JS
// Template literals
let name = "Alice";
let age = 30;
console.log(`My name is ${name}
and I am ${age} years old.`);

// Destructuring
const person = { name: "Bob", age: 25 };
const { name: pName, age: pAge } = person;
console.log(pName, pAge);

// Spread operator
let arr1 = [1, 2, 3];
let arr2 = [...arr1, 4, 5];
console.log("Spread:", arr2);

// Rest parameters
const sum = (...nums) =>
    nums.reduce((a, b) => a + b, 0);
console.log("Sum(1,2,3,4):", sum(1, 2, 3, 4));

// Optional chaining
const user = { profile: { email: "a@b.com" } };
console.log(user?.profile?.email);
console.log(user?.address?.city); // undefined, no error
